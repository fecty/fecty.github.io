# zurain.github.io

## Overview

This is Zurain's portfolio website, showcasing skills, services, and contact information. The site is built using HTML and CSS, organized for clarity and maintainability.

## Project Structure

```
index.html                # Main landing page
README.md                 # Project documentation
assets/                   # Images, icons, and other static assets
pages/                    # Individual site pages
	├── about.html          # About section
	├── construction-page.html # Under construction page
	├── contact.html        # Contact information
	├── services.html       # Services offered
	└── skills.html         # Skills showcase
styles/                   # CSS stylesheets
	└── style.css           # Main stylesheet
```

## How to Use

Open `index.html` in your browser to view the site. Navigate to other pages via the menu or direct links.

## Customization

- Update content in the HTML files under `pages/` as needed.
- Add images or icons to the `assets/` folder.
- Modify styles in `styles/style.css` for design changes.
